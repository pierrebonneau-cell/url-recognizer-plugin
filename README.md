# URL Recognizer Plugin

This is a plugin for Obsidian (https://obsidian.md).

This project uses TypeScript to provide type checking and documentation.
The repo depends on the latest plugin API (obsidian.d.ts) in TypeScript Definition format, which contains TSDoc comments describing what it does.

This plugin facilitates the creation of URLs from configurable textual patterns.

The default shortcut is Ctrl+Space.
It must be invoked either with the whole text to replace selected, or with the cursor at the end of a text to replace.
